# TFM2 Forge

**A visual champion creator for Teamfight Manager 2.** Design a champion's stats,
abilities, art, effects and sounds in a friendly desktop app — TFM2 Forge writes a
complete, valid mod straight into your game's `mods/` folder. No JSON, no coding.

*Created by **Zietes**. Fan-made; not affiliated with TeamSamoyed.*

## Download

➡️ **[Download the latest release](https://github.com/Zietes/tfm2-forge/releases/latest)**

- **`TFM2Forge-Setup.exe`** — installer (recommended; per-user, no admin needed)
- `TFM2Forge-Alpha.zip` — portable version (extract & run)

**Requirements:** Windows 10/11 and Teamfight Manager 2 (Steam). Everything else is
bundled — no Python or browser needed.

> This is an **unsigned Alpha** build, so Windows SmartScreen may warn you — click
> **More info → Run anyway**. Verify your download against `SHA256SUMS.txt`.

## What it does

- **Guided + advanced (node *Flow*) ability editor** with 36 ready-made recipes
- **Live Ability Preview** — draws range / area / timing to scale
- **Balance & Feedback** — ranks your champion against the 60 base champions
- Custom **art, projectiles, effects and sounds**; multi-champion mods
- **Live validation** so a champion is correct before it ever hits the game
- **One-click auto-update** — the app checks here and updates itself

It reads art and data from **your own** installed copy of the game; nothing from
Teamfight Manager 2 is bundled or redistributed.

## Updates

TFM2 Forge checks this repository's releases on launch and can install updates in
one click. You can turn the check off in the app's **? Help** box.
